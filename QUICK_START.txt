STEP 1: Open Port 8006 in AWS Security Group (CRITICAL!)
────────────────────────────────────────────────────────
Without this, the mobile app CANNOT connect to the backend!

1. Go to: AWS Console → EC2 → Security Groups
2. Select your instance's security group
3. Inbound Rules → Edit Inbound Rules → Add Rule
4. Configure:
   Type:        Custom TCP
   Port:        8006
   Source:      0.0.0.0/0
   Description: Zelux Backend API

5. Save Rules

Test from your laptop:
  curl http://3.24.31.8:8006/health
  
Expected response:
  {"status":"healthy","service":"zelux-api"}
